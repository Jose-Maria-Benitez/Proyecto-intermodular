Componentes del equipo informático

A continuación, se describirán los principales componentes de los ordenadores que formarán parte de este entorno de trabajo a rasgos generales.

CPU

La Unidad Central de Procesamiento es fácilmente una de las partes más importantes del ordenador. Se encarga de ejecutar e interpretar las instrucciones de los programas, además de coordinar el software del ordenador. Por tanto, podríamos decir que se trata del cerebro del dispositivo.

Es uno de los cuatro pilares de la arquitectura Von Neumann, caracterizada por unificar instrucciones y datos en una sola memoria principal.

Los componentes de la CPU son:

  • Unidad de control: parte encargada de dirigir todas las operaciones (recoge las instrucciones, las decodifica y las ejecuta).
  • Unidad aritmético-lógica (ALU): calculadora que, como su nombre indica, se encarga de realizar operaciones matemáticas y lógicas.
  • Registros: almacenan las operaciones realizadas por el procesador basándose en bancos de registros o elementos de memoria SRAM (static random access memory, la cual no necesita refrescarse para actualizarse).
  • Reloj: elemento que marca la velocidad de los ciclos de instrucción.

Placa base

Se trata de la plataforma sobre la que se constriurá todo el sistema. Su función es dar soporte físico y distribuir la energía y los datos a cada parte del ordenador. Por tanto, si decíamos que la CPU era el cerebro del ordenador, la placa base es el esqueleto y el sistema nervioso. Es muy importante saber elegirla correctamente porque de ella dependerá:

  • La CPU: en la placa base encontraremos el zócalo de la CPU. Cada familia de procesador tiene un zócalo específico, por lo que deberemos escoger una placa base que sea compatible con nuestro procesador.
  • La RAM: el número (generalmente 2 o 4 canales) y el tipo de RAM que soportan dependerán de las ranuras que traiga la placa base.
  • Ranuras de expansión: las funcionalidades que podamos añadir al ordenador dependerá de estos conectores. Un ejemplo de esto es el PCIex16 para la tarjeta gráfica.
  • Almacenamiento: también dependerá de la placa base los discos duros que podremos conectar mediante según qué conectores: el SATA para discos más tradiciones (que no usaremos para este proyecto), o el M.2 para SSDs más rápidos y compactos.
  • Chipset: parte de la placa que gestiona el flujo de datos entre todos los componentes y que determinará las medidas de la placa.

Memoria RAM

Random access memory, es el componente encargado de almacenar de forma temporal tanto las instrucciones del programa que se está ejecutando como los datos que este necesita para operar. Es una memoria volátil, lo que significa que su contenido se borra cuando el ordenador se apaga.

Para aprovechar al máximo su potencial, en caso de que la placa base cuente con ranueras dual channel, instalaremos las RAM en los puertos pares o impares (1 y 3, o 2 y 4) para evitar cuellos de botella a la hora de mover los datos, garantizando un tráfico más fluido y rápido.

Discos de almacenamiento

Estos discos son más lentos que la RAM, pero almacenan la información de forma permanente.

Durante mucho tiempo el estándar era el HDD, un dispositivo mecánico con platos giratorios y un cabezal lector que se mueve físicamente para encontrar la información.

Tras ellos llegaron los discos SSD, que al no depender de piezas mecánicas, los SSD ofrecen tiempos de acceso casi instantáneos, una lectura/escritura mucho más rápida, cero ruido y gran resistencia física.

Fuente de alimentación

Este componente suministra energía al ordenador y la adapta para que todos los demás componentes funcionen correctamente, transformando la corriente alterna a corriente continua y encargándose de que cada componente recibe el voltaje adecuado.

Tarjetas de expansión

Las tarjetas de expansión, como se dijo anteriormente, se conectan a la placa base y añaden funcionalidades extendidas al ordenador. Para el propósito de este proyecto, contaremos una el PCIe x16 para añadir una mejor tarjeta gráfica (GPU) a los ordenadores de la sala de seguridad, y para los de recepción, con el PCIe x4 y x1 para añadir funcionalidades de sonido (por temas de comunicación) y Wifi (por temas de conectividad).
