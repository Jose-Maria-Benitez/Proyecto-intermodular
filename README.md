# Proyecto-intermodular
<h1>Proyecto intermodular ASIR  primer año</h1>
<h2>Por José María Benítez Foncubierta</h2>
<p>Este proyecto simulará la adquisición de un nuevo crucero de tamaño pequeño (500 tripulantes aproximadamente) por parte de una empresa de turismo. Por consiguiente se procederá a montar los equipos informáticos básicos para el navío, así como la implementación de sus correspondientes sistemas operativos para algunas zonas del barco (recepción, zonas de ocio...); su propia base de datos, la cual será proveída desde un servidor en la nube; y se diseñará y aplicará su tipología de red.</p>
<p>De este modo, el ptoyecto tendrá en cuenta los siguientes usuarios:</p>
<ul>
<li>Trabajadores de recepción</li>
<li>Trabajadores de la zona de seguridad</li>
<li>Trabajadores de otras zonas del barcola zona de seguridad</li>      
<li>Los propios clientes</li>      
</ul>
<p>De este modo, se buscará el equipo informático más asequible y apropiado para las tareas que se realizarán en cada una de las zonas, con sus correspondientes sistemas operativos; se creará una base de datos con sus correspondientes permisos y roles para guardar la información de la empresa de forma segura; y se habilitará una red para los trabajadores y para los clientes de manera que todo el barco quede comunicado.</p>
